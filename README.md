# HR Performance Analysis Dashboard

## 📌 Project Overview
This project presents an interactive Tableau dashboard analyzing employee performance, workforce demographics, attrition trends, and departmental insights within an organization.

The dashboard helps uncover patterns in employee distribution, attrition rates, job satisfaction, and workforce performance metrics to support data-driven HR decision-making.

---

## 🎯 Objectives
- Analyze employee attrition trends
- Evaluate workforce demographics
- Monitor department performance
- Track employee satisfaction metrics
- Identify patterns in employee distribution
- Support HR decision-making with data insights

---

## 🛠️ Tools Used
- Tableau
- Excel / CSV

---

## 📊 Dashboard Features
The dashboard includes:

- Employee demographic analysis
- Attrition analysis
- Department-wise performance insights
- Workforce distribution visualization
- Interactive filters and charts
- KPI metrics for HR analysis

---

## 📈 Key Insights
- Certain departments experienced higher attrition rates
- Workforce distribution varied significantly across departments
- Employee demographics revealed notable workforce patterns
- HR metrics highlighted areas requiring management attention
